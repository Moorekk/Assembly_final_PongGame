# Ping pong game

> 2 人乒乓球遊戲

## 操作方法
左邊玩家以鍵盤 $$W,\ S$$ 控制上下  
右邊玩家以鍵盤 $$O\ ,\ L$$ 控制上下  


## 啟用方法
- **僅能在 Windows 系統使用**  
- 執行前須在terminal 輸入```chcp 437``` 更改活動主控台字碼頁為美國編碼。  
- 執行 ```fianl_ballgame.exe```
